# Geb

A simple mobile-friendly website.

## Setup

Simply open `index.html` in your browser to view the site.

## Structure

- `index.html` - Main HTML file
- `styles.css` - Stylesheet with mobile-optimized design
- Mobile-first responsive design with viewport meta tag

## Features

- Blue background (#4A90E2)
- Centered content optimized for mobile screens
- Clean, minimal design ready for future expansion
